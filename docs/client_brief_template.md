# Client Brief Template

Ask the client to answer:

1. What exact problem are you facing now?
2. What final result do you want?
3. What materials already exist?
4. What output format do you need: table, report, script, page, PDF?
5. When do you need delivery?
6. What budget range do you have?
7. What must not be touched?
8. If this works, will the same need repeat?
