# 中文外联包说明

这组文件是给最现实、最容易出第一笔钱的渠道准备的：

1. 熟人
2. 微信群
3. 一对一私聊

## 文件列表

1. `cn_outreach_packets/微信群首发短文案.txt`
2. `cn_outreach_packets/一对一私聊首发.txt`
3. `cn_outreach_packets/对方有兴趣后怎么接.txt`
4. `cn_outreach_packets/对方先问价格怎么接.txt`
5. `cn_outreach_packets/48小时未回复跟进.txt`

## 使用顺序

1. 先发群里或一对一首发
2. 对方有兴趣，就发“有兴趣后怎么接”
3. 对方直接问价格，就发“先问价格怎么接”
4. 48 小时没回复，再发一次轻跟进

## 原则

1. 不一上来发长 proposal
2. 不把话说太满
3. 先把对方带到“把需求发过来”的动作上
