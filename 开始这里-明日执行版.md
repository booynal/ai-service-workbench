# 开始这里：明日执行版

如果明天你只想做最少动作，又不想在一堆文件里找来找去，就按下面走。

如果你想先知道“下一条最该发谁”，先跑：

```bash
python3 tools/next_outreach.py 触达计划.csv 触达日志模板.csv
```

如果你想连这一步都省掉，直接跑：

```bash
python3 tools/today_actions.py
```

如果你担心发之前漏了关键文件，再跑：

```bash
python3 tools/preflight_check.py
```

如果你想直接一条命令把今天该发的 runbook 生成出来，就跑：

```bash
python3 tools/start_today.py tmp_runbooks/today.md
```

## 1. 中文高概率渠道

先跑：

```bash
python3 tools/render_outreach_runbook.py wechat-group runbooks/wechat-group.md
```

然后直接打开：

`runbooks/wechat-group.md`

如果你只是想把入口先丢出去，不想先发长文本，也可以直接发中文落地页：

`https://booynal.github.io/ai-service-workbench/zh.html`

如果你更适合直接发图，现在也已经有两张现成素材：

1. `share_assets/zh-landing-qr.png`
2. `share_assets/wechat-share-card.png`

这份文件已经包含：

1. 首发文案
2. 对方有兴趣怎么接
3. 对方问价格怎么接
4. 48 小时没回复怎么跟
5. 发完以后怎么记一笔

## 2. 英文陌生外联

如果你想试一次英文陌生外联，跑：

```bash
python3 tools/render_outreach_runbook.py csa-issue-2 runbooks/csa-issue-2.md
```

然后直接打开：

`runbooks/csa-issue-2.md`

## 3. 发完以后记一笔

中文群/私聊：

```bash
python3 tools/log_outreach_entry.py 触达日志模板.csv 2026-06-03 wechat "你的对象" "cn_outreach_packets/微信群首发短文案.txt" sent "wait reply" "first outreach"
```

英文陌生外联：

```bash

```

## 4. 明天不要做的事

1. 不要继续改脚本
2. 不要继续做页面
3. 不要同时开很多平台

## 5. 明天最少完成线

只要做成下面 2 件事，就算从“准备阶段”进入“真实跑业务”：

1. 发出至少 3 次真实触达
2. 在 `触达日志模板.csv` 里留下真实记录
