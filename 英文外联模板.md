# 英文外联模板

## 1. GitHub / Email / DM 冷启动模板

```text
Hi,

I came across your issue and it looks close to the kind of automation / workflow / delivery work I already handle.

I help turn repetitive manual work into:
1. automation scripts
2. structured reports
3. lightweight AI-assisted workflows

I already drafted a first-pass proposal structure for similar requests on my side, so if you want, I can send a short version of:
1. what I think the scope is
2. what a practical first deliverable would be
3. what information is still missing

No pressure. If useful, I can keep it concise.
```

## 2. 对对方已有 issue 的跟进模板

```text
Hi,

Your issue reads like a real delivery problem rather than just a technical note, so I took a closer look.

My rough take is:
1. this should probably be handled as a scoped implementation task
2. the fastest path is a small first version, not a broad redesign
3. the missing details are likely around inputs, environment, and expected outputs

If helpful, I can share a short suggested delivery path.
```

## 3. 对报价敏感对象的模板

```text
Hi,

If budget is a concern, the best path is usually to start with a small first version that proves the workflow and avoids overbuilding.

That way you can validate:
1. whether the approach works
2. whether the output is actually useful
3. whether it is worth extending further
```
