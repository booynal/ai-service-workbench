# 首周落地清单

## Day 1

1. 确定主卖点：AI 代做服务，不卖空概念
2. 整理 3 个标准套餐
3. 生成 1 份样例提案
4. 写出 1 版对外发布文案

## Day 2

1. 把样例放到 GitHub 仓库或 Gist
2. 准备 3 个案例标题
3. 选 2 个渠道先发：微信/闲鱼 或 GitHub/社群

## Day 3

1. 针对收到的需求，用本地提案工具快速回复
2. 记录客户常见问题
3. 修正套餐边界和报价

## Day 4-7

1. 至少发出 20 次有效触达
2. 把高频需求归类
3. 形成 1 个最常卖的标准交付模版
4. 把最常见服务进一步半自动化

## 核心指标

第一周只看这几个数：

1. 发出多少次有效触达
2. 收到多少次真实咨询
3. 有多少次进入报价阶段
4. 成交了几单
5. 哪个服务最容易成交
